THE ZILOG Z80 (Z80) — a UD0 EXEREÚNESIS teardown + spec edition. Designed by Federico Faggin's Zilog, the Z80 was an 8-bit CPU compatible with the Intel 8080 but with more registers and instructions — and it became one of the longest-lived processors ever. It ran CP/M computers, countless arcade boards, the Master System, the Game Gear, and the MSX standard.
